# Sprint announcement examples

## Sprint kickoff example

```markdown
:rocket: **Sprint Engagement 2025-26**

Hello tout le monde :wave:,

Le nouveau sprint **2025-26** de l'équipe **Engagement** vient d'être lancé :

:date: **Dates de sprint** : Du 15 décembre au 26 décembre 2025 (durée 2 semaines)

:dart: **Objectif(s) du sprint** :
:one: Améliorer la conversion sur l'écran de paiement
:two: Réduire le nombre de tickets support liés à l'onboarding

:sparkles: **Contenu du sprint** :

**Nouvelle fonctionnalité X** :
- Ajouter le de validation en 3 étapes
- Intégrer le module de notifications
- Corriger les problèmes de scroll sur mobile

**Bugs** :
- Résoudre l'erreur 500 sur la page de profil
- Corriger l'affichage des avatars sur Safari

**Tech** :
- Migrer vers la nouvelle API REST
- Optimiser les requêtes de la liste principale

**Autres** :
- Mettre à jour la documentation utilisateur
- Revue de code sur le module authentification

:warning: Important : Les tickets inclus dans chaque sprint ont pour objectif d'être validés d'ici la fin de celui-ci. Cependant, certaines mises en production peuvent être légèrement décalées en fonction des dépendances et des priorités.

Si vous avez des questions ou remarques, n'hésitez pas à répondre dans le fil de discussion !

Bonne journée :panda_dance:
```

## Release announcement example

```markdown
:rocket: Team invitations are now 3x faster

Hello everyone :wave:,

Why: Eliminate friction when inviting a teammate — every lost invitation is a missed collaboration

How: We redesigned the invite flow from 5 steps to a single screen

What changed for users:
- Invitation in 1 click (before: 5 steps)
- Instant email confirmation
- Customizable roles from the first screen

Who is affected:
- All workspace owners and admins

Rollout:
- 100% starting today

Need help?
- Updated FAQ: help.example.com/invites
- Contact: support@example.com

Have a good day :panda_dance:
```
