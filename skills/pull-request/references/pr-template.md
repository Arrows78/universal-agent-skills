# Merge request templates

## Standard PR

```markdown
## Summary

[What this PR changes and why.]

## Changes

- [Change 1]
- [Change 2]

## How to test

1. [Precondition]
2. [Action]
3. [Expected result]

## Screenshots or recordings

[N/A]

## Related issues

- Refs #[id]

## Deployment and rollout notes

[N/A]
```

## Small fix

```markdown
## Summary

[One or two sentences.]

## How to test

- [Validation step]

## Related issues

- Fixes #[id]
```

## Large feature

```markdown
## Summary

[What changed, why it matters, and who is affected.]

## Approach

[Key design or implementation choices.]

## Changes

- [Change 1]
- [Change 2]

## How to test

1. [Step]
2. [Step]

## Risks and mitigations

| Risk | Mitigation |
| --- | --- |
| [Risk] | [Mitigation] |

## Deployment and rollout notes

- Feature flag: [flag]
- Migration: [yes/no]
- Monitoring: [dashboard/logs]
```
