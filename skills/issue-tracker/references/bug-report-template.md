# Bug report template

## Copy-paste template

```markdown
📚 **CONTEXTE**

- Courte description pour comprendre rapidement le problème
- [Hypothèse, contournement, regression suspectée, ou "Aucune note"]

📝 **COMMENT REPRODUIRE**
1. [Étape 1]
2. [Étape 2]
3. [Étape 3]

❌ **COMPORTEMENT ACTUEL**
- [Résultat observé]
- [Message d'erreur, capture, log, lien si disponible]

✅ **COMPORTEMENT ATTENDU**
- [Résultat attendu]
- [Comportement de référence]
```

## Field guidance

- `📚 CONTEXTE`: include enough details to quickly understand the issue without exposing secrets.
- `📝 COMMENT REPRODUIRE`: one action per line, in order.
- `❌ COMPORTEMENT ACTUEL`: observable facts only.
- `✅ COMPORTEMENT ATTENDU`: expected product behavior, not implementation assumptions.
